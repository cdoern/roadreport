# RoadReport

Crowdsourced road and sidewalk condition map for runners, walkers, cyclists, and commuters.

For full setup instructions, see **[specs/001-road-report-heatmap/quickstart.md](specs/001-road-report-heatmap/quickstart.md)**.

## Tech Stack

| Layer       | Technology                              |
|-------------|------------------------------------------|
| Frontend    | Vite 5 + React 18 + TypeScript 5        |
| Styling     | Tailwind CSS 3                           |
| Map         | react-leaflet v4 + leaflet.heat          |
| Backend     | Supabase (PostgreSQL 15 + PostGIS 3)    |
| Hosting     | Vercel Hobby                             |
